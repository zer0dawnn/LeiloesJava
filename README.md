# Projeto Leilões TDSat

## 📖 Sobre o Projeto
Este projeto é um sistema de leilões desenvolvido em Java para auxiliar a Casa de Leilões na gestão de produtos e lances. O objetivo é informatizar o processo de vendas e arremates.

## 🛠️ Tecnologias Utilizadas
* **Java**: Backend e lógica de negócio.
* **MySQL**: Banco de dados.
* **IDE NetBeans**: Ambiente de desenvolvimento.
